# Sort file

Sorts a file using R [`sort()`](https://rdrr.io/r/base/sort.html).

## Usage

``` r
sort_file(filename)
```

## Arguments

- filename:

  file to be sorted

## See also

[`sort_rbuildignore()`](https://jabenninghoff.github.io/rdev/reference/sort_rbuildignore.md)

## Examples

``` r
if (FALSE) { # \dontrun{
sort_file(".Rbuildignore")
} # }
```
